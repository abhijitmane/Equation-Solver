SIMPLEX METHOD

 




 * Standard Form of LP model: 

        max z = c1*x1 + c2*x2 + ........... + cn*xn
        subject_to :
		a11*x1 + a12*x2 + ........+ a1n*xn <= b1
		a21*x1 + a22*x2 + ........+ a2n*xn <= b2       
		.
		.
		am1*x1 + am2*x2 + ........+ amn*xn <= bm
		x1,x2,....,xn >= 0

AND

        min z = c1*x1 + c2*x2 + ........... + cn*xn
        subject_to :
		a11*x1 + a12*x2 + ........+ a1n*xn >= b1
		a21*x1 + a22*x2 + ........+ a2n*xn >= b2       
		.
		.
		am1*x1 + am2*x2 + ........+ amn*xn >= bm
		x1,x2,....,xn >= 0

Whenever the inequality in constraints is <= , the slack variables are added and whenever the
inequality in constraints is >= , the surplus variables are added in order to get equations.
The slack variables act as the initial basic feasible solution to the model.
Whenever the slack variables are not in the model , the artificial variables are added in order
to get the initial basic feasible solution to the model.
	Here(in case of artificial variables) the simplex big-M method is used to get the optimal solution.
You can see Solved Problem(all Iterations) on Simplex Tab.
