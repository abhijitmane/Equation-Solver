DUALITY

 

 * Relationship between PRIMAL & DUAL:

           PRIMAL             -          DUAL

1. # of constraints           -    # of variable
2. # of variables             -    # of constraints
3. RHS                        -    Objective Function
4. Objective Function         -    RHS
5. Coeff. matrix(A)           -    Transpose of Coeff. Matrix(A^t)
6. Equality(=) Constraints    -    Unrestricted Variables
7. Unrestricted Variables     -    Equality(=) Constraints
8. <= Constraints             -    >= Variables
9. >= Constraints             -    <= Variables
10.<= Variables               -    <= Constraints
11.>= Variables               -    >= Constraints
12. MAX Problem               -    MIN Problem 

 General View:

	If the primal is,
		max z = CX
		subject_to :
			AX <= b
			Xi>=0
		where C - Coefficient matrix in the objective function.
		      A - Coefficient matrix of the decision variables in the constraints.
		      X - Decision variable matrix.
		      
	The dual form of above primal problem is,
		min z = bX'
		subject_to :
			A'X' >= C
			Xi'>=0
		where X' - new decision variables.
		      A' - Transpose of matrix A
