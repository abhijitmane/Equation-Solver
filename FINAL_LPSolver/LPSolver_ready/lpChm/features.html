INTRODUCTION

WELCOME to LPSolver... 
 



	
                      LPSolver is a software tool, uesd for solving Linear Problems.
              Most of the real life problems are LP Problems(like estimating proffit or loss,
              avoiding producer consumer problems etc.)

      Types of Linear Programming models :
	    1. Maximization of objective function.
	    2. Minimization of objective function.
        	Min Z = Max -Z

         a)SIMPLEX METHOD:
               Method to solve Linear programming models(used Big M method as per the requirement)
           some special termination cases:
              |_ Unbounded solution space & Unbpunded Objective value 
              |_ Alternate Solution to LP
              |_ Infeasibility etc.

         b)PRIMAL TO DUAL:
             You can find Duality of problems(Initial problem is called as Primal)
      
         c)SENSITIVITY ANALYSIS:
            case1: Range of Values for Objective Function Coefficients of NBVs.
            case2: Range of Values for Objective Function Coefficients of BVs.
            case3: Range of Values for RHS values of Constraints.
            case4: Range of Values for Constraint Coefficients of NBVs.
            case5: Add a New Decision Variable.

         NOTE: For each special case, sample examples are provided.
